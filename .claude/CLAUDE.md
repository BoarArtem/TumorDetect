# Tumor Detect

Tumor Detect - веб-приложение на базе ИИ, которое поможет нейрохирургам быстро посмотреть наличие и расположение опухоли на основе МРТ снимка с помощью сегментации опухоли

## Goal

Проект нужен для медиков, нейрохирургов с целью быстрого анализа МРТ снимка на наличие опухоли мозга
